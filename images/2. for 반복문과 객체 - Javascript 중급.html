    2. for 반복문과 객체

    객체와 함께 사용하는 for of ,  for in,  for each

     객체란?

    - 객체는 자료를 저장하고 처리하는 기본단위 (cf. 원시타입(변수 : 값 = 1:1))

    - 자바스크립트 프로그램에서 원시타입을 제외한 복합자료형

    - 자바스크립트는 객체 기반 프로그래밍 언어로서 다양한 객체가 존재

    - 객체에는 고유한 속성인 프로퍼티(Property)와 기능인 메서드(Method)가 있음.


    const 배열 =[‘바나나’,‘사과’,‘귤’];

    1. for of 문

    반복 가능(iterable)한 객체의 값을 반복 (객체:  배열, 문자열, map, NodeList....)

    [기본형] for (const 요소 of   객체) {}

    const cars = ["BMW", "Volvo", "Mini"];

    let text = "";
    for (let x of cars) {
      text += x;
          console.log(x);  
    }


    실행 언어: html

    2. for in

    [기본형] for (const 인덱스 in 객체){ }


    const person = {fname:"John", lname:"Doe", age:25};
    let text = "";
    for (let x in person) {
      text += person[x];
      document.write(person[x] +","); //john,Doe,25, 출력
    }

    3. for each 

    forEach()메서드는 배열의 각 요소에 대한 콜백함수를 호출한다.

    [기본형] 배열.forEach(function(요소, 인덱스, 배열명){}

    * 콜백함수 : 매개변수로 함수를 호출하는 함수

    let  sum = 0 ;
    const  numbers =[65, 44, 12, 4];
    numbers.forEach (myFunction);

    function  myFunction(value) {
      sum += value;
    }

    const  numbers =[65, 44, 12, 4];
    numbers.forEach (myFunction2)

    function  myFunction2(value, index, arr) {
      arr[index] = value* 10 ;
    }

    인덱스, 배열명은 생략 가능함.
